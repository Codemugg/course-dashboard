BLOCK LEVEL TAGS (Part - I):


Each Block Level Tag element in an HTML document are displayed on a new line.

BLOCK LEVEL TAGS:
  - Paragraph tag <p> defines paragraph.
  - HTML defines six levels of headings.
    The heading element are h1, h2, h3, h4, h5, and h6 with h1 being the highest level and h6 the least level.
  - The <br> tag inserts for a single line break.

HTML HEADINGS :

Using html heading tags highlights the text information written inbetween the tags.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     <h1> This is heading displayed using h1 tag. </h1>
|     <h2> This is heading displayed using h2 tag. </h2>
|     <h3> This is heading displayed using h3 tag. </h3>
|     <h4> This is heading displayed using h4 tag. </h4>
|     <h5> This is heading displayed using h5 tag. </h5>
|     <h6> This is heading displayed using h6 tag. </h6>
|  </body>
| </html>

OUTPUT:

 

This is heading displayed using h1 tag.

 

This is heading displayed using h2 tag.

 

This is heading displayed using h3 tag.

 

This is heading displayed using h4 tag.

 

This is heading displayed using h5 tag.

 
This is heading displayed using h6 tag.


HTML PARAGRAPH :

The Paragraph tag defines a paragraph in the html document.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     <p>This is my Paragraph. </p>
|  </body>
| </html>

OUTPUT:

|  This is my Paragraph.



HTML BREAK :

The break tag ( <br> ) inserts a single line break.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     This is FIRST LINE. <br>
|     This is SECOND LINE. <br>
|  </body>
| </html>

OUTPUT:

|     This is FIRST LINE.
|     This is SECOND LINE.


BLOCK LEVEL TAGS (Part - II):


BLOCKQUOTE :

The <blockquote> tag specifies a section that is quoted from another source. Browsers usually indent <blockquote> elements.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     <blockquote>
|     <p> This is an example of blockquote tag.
|     Here is a long quotation. </p>
|     </blockquote>
|  </body>
| </html>

OUTPUT:


This is an example of blockquote tag. Here is a long quotation.




PRE :

The <pre> tag used for indicating preformatted text.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     <pre> This is preformatted text. </pre>
|  </body>
| </html>

OUTPUT:

 |   This is preformatted text.


ADDRESS :

The <address> tag provides contact information for a document. The text in the <address> element usually renders in italic. This information includes name, email address, postal address, and phone number, etc.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     Address: <address> abc@xyz.com </address>
|  </body>
| </html>

OUTPUT:

|     Address:
abc@xyz.com



HORIZONTAL RULE:

The <hr> tag represents a horizontal rule. The <hr> tag is used to separate content in an HTML page. It has align, no shade, size, width and color attribute.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     This is FIRST LINE. <hr>
|     This is SECOND LINE. <hr>
|  </body>
| </html>

OUTPUT:

|   This is FIRST LINE.
|   This is SECOND LINE.
