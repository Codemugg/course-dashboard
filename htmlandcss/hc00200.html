HTML AND CSS COURSE FOR BEGINNERS..!

HTML TAGS:


All HTML documents start with a document type declaration: <!DOCTYPE html>.
The HTML document itself begins with <html> and ends with </html>.

HTML document is divided into two parts: The head and the body. The <head> element is used to specify information about the HTML Web Page as title of the Web Page. The actual content of the Web Page are specified inside the <body> element.

For Example:
1. Open a TEXT EDITOR.
2. Write the following HTML codes to display sample message.

NOTE: Save the file using '.html' extension.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     HTML and CSS course for BEGINNERS by CODEMUGG.
|  </body>
| </html>

OUTPUT:



!DOCTYPE html TAG:


Every html document should start with a <!DOCTYPE html> tag.

It is an "information" to the browser about what document type to expect.

| <!DOCTYPE html>

html TAG:


The tag represents the root of an HTML document.

The tag is the container for all other HTML elements.

|  <html>
|  
|  </html>


head TAG:


The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.

HTML metadata is data about the HTML document. Metadata is not displayed.

Metadata typically define the document title, character set, styles, scripts, and other meta information.

|  <head>
|   <!-- Title, links, meta tag, styles -->
|  </head>

NOTE : By the way, We use "<!--Comments-->" tag to comment in an html document..!



title TAG:


The <title> element defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.

The <title> element:

    a. defines a title in the browser toolbar,
    b. provides a title for the page when it is added to favorites,
    c. displays a title for the page in search engine-results.

|  <head>
|   <title>Title_of_Page</title>
|  </head>


body TAG:


The <body> tag defines the document's body.

The <body> element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

You'll study the above mentioned contents in detail later in this course.

NOTE : There can only be one element in an HTML document.

|  <body>
|   Contents of Document.
|  </body>
