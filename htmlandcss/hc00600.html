HTML AND CSS COURSE FOR BEGINNERS..!

LISTS :


You can use HTML list to group few items together inside a single title.

The lists are broadly classfied in Ordered List and Unordered List.

Following block shows the example of an Unordered list and a ordered list.

|     Ordered List:         Unordered List:
|       First_element         Element
|       Second_element        Element
|       Thrid_element         Element
|       Fourth_element        Element
|       Fifth_element         Element

UNORDERED LIST :


An unordered list consists of element which are not sorted in a proper ordered.

It lists item using plain bullets, circle is the default bullet used.

A <ul> tag represents an unordered list and <li> represents item present in the list.

| <head>
| </head>
| <body>
| <h3>Subjects</h3>
| <ul>
|   <li>Mathematics</li>
|   <li>Physics</li>
|   <li>Language</li>
|   <li>Chemistry</li>
| </ul>
| </body>

Subjects

  • Mathematics
  • Physics
  • Language
  • Chemistry

ORDERED LIST :


An ordered list consists of element which are arranged in a particular order.

It lists item using Numbers and alphabets, numbers is the default bullet used.

A <ol> tag represents an ordered list and <li> represents item present in the list.

| <head>
| </head>
| <body>
| <h3>Subjects</h3>
| <ol>
|   <li>Mathematics</li>
|   <li>Physics</li>
|   <li>Language</li>
|   <li>Chemistry</li>
| </ol>
| </body>

Subjects

  1. Mathematics
  2. Physics
  3. Language
  4. Chemistry

DEFINITION / DESCRIPTION LIST :


A definition list arranges items in the same way as they are arranged in a dictionary.

It lists item using Numbers and alphabets, numbers is the default bullet used.

A <dl> tag represents definition list, <dt> represents terms(name/title) and <dd> represents description of title.

| <head>
| </head>
| <body>
| <h3>Subjects</h3>
| <dl>
|   <dt>Mathematics</dt>
|     <dd>Consits of Algebric Equations, Geometry and Statistics.<dd>
|   <dt>Science</dt>
|     <dd>Consits of Physics, Biology and Chemistry.<dd>
| </dl>
| </body>

Subjects

Mathematics
    Consists of Algebric Equations, Geometry and Statistics.
Science
    Consists of Physics, Biology and Chemistry.


HTML AND CSS LIST:


The CSS list properties allows you to:

  - Set different list item markers for ordered lists

  - Set different list item markers for unordered lists

  - Add background colors to lists and list items

LIST ITEM MARKERS:


Ordered List:

You can set different list item marker using list-style-type property of CSS.

Following example shows the available ordered list markers in CSS.

| <head>
| <style>
|   ol{
|     list-style-type: lower-roman;
| }
| </style>
| </head>
| <body>
| <h3>Subjects</h3>
| <ol>
|   <li>Mathematics</li>
|   <li>Physics</li>
|   <li>Language</li>
|   <li>Chemistry</li>
| </ol>
| </body>

Subjects

  1. Mathematics
  2. Physics
  3. Language
  4. Chemistry
The above code sets the list-style-type property to value lower-roman.

You can also try following type:
decimal-leading-zero
lower-greek
lower-latin

Unordered List:

Following example shows the available unordered list markers in CSS.

| <head>
| <style>
|   ul{
|     list-style-type: disc;
| }
| </style>
| </head>
| <body>
| <h3>Subjects</h3>
| <ul>
|   <li>Mathematics</li>
|   <li>Physics</li>
|   <li>Language</li>
|   <li>Chemistry</li>
| </ul>
| </body>

Subjects

  1. Mathematics
  2. Physics
  3. Language
  4. Chemistry
The above code sets the list-style-type property to value disc.

You can also try following type:
circle
square

ADDING BACKGROUND COLOR:


We can also style lists with colors, to make them look a little more interesting.

Anything added to the <ol> or <ul> tag, affects the entire list, while properties added to the <li> tag will affect the individual list items:

| <head>
| <style>
|   ul{
|     background: #ec3750;
| }
|   ul li{
|     background: #e28e9a;
|     color: #000000;
| }
| </style>
| </head>
| <body>
| <h3>Subjects</h3>
| <ul>
|   <li>Mathematics</li>
|   <li>Physics</li>
|   <li>Language</li>
|   <li>Chemistry</li>
| </ul>
| </body>
