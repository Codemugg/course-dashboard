Now, Lets move on to CSS.

CSS


As mentioned earlier CSS is a stylesheet which is used to style the visual presentation of web pages.

CSS SYNTAX :

CSS syntax includes selector, properties, values, declarations, declaration blocks, rulesets, at-rules, and statements.

SELECTOR : A selector is a code snippet used to identify the web page element or elements that are to be affected by the styles.

PROPERTY : A property is the aspect of the element that is to be affected. For example, color, padding, margin, and background are some of the most commonly used CSS properties.

VALUE : A value is used to define a property. For example, the property color might be given the value of red for example: color : red;.

DECLARATION : The combination of a property and a value is called a declaration.

DECLARATION BLOCKS : In many cases, multiple declarations are applied to a single selector. A declaration block is the term used to refer to all the declarations applied to single selector.

RULESET : A sigle selector and the declaration block that follows it in combination are referred as a ruleset.

AT-RULES : At-rules are similar to rulesets but begin with the '@' sign rather that with a selector . The most common at-rule is the @media rule which is often used to create a block of CSS rules that are applied based on the size of the device viewing the webpage.

Example of CSS Syntax :


Let's use a block of CSS to clarify what each of these items is.

| h1 {
| color: red;
| font-size: 3em;
| text-decoration: underline;
| }

In this example, h1 is the selector. The selector is followed by a declaration block that includes three declarations. Each declaration is separated from the next by a semicolon. The tabs and line breaks are optional but used by most developers to make the CSS code more human-readable. By using h1 as the selector, we are saying that every level 1 heading on the web page should follow the declaration contained in this ruleset. The ruleset contains three declarations:

  • color: red;
  • font-size: 3em;
  • text-decoration: underline;


color, font-size, and text-decoration are all properties. There are literally hundreds of CSS properties that you can use, but only a few are commonly used. The color property can use a color keyword or a color formula in Hex, RGB, or HSL format. In this case, the color keyword red. There are a few color keywords available in CSS3, but millions of colors can be accessed with the other color models.

The value 3em to the property font-size. There are a wide range of size units we could have used including pixels, percentages, and more. Finally, the value underline to the property text-decoration. The overline or line-through can be used as values for text-decoration. In addition, CSS3 allows for the use of the line-styles solid, double, dotted, dashed, and wavy was well the specification of text-decoration colors.

Ways of Linking CSS Rules to an HTML Document :


Earlier we studied there are three ways of adding CSS rules to a web page:
  • INLINE CSS
  • INTERNAL CSS
  • EXTERNAL CSS

Inline Styles :

Inline styles are applied to specific HTML elements. The HTML attribute style is used to define rules that only apply to that specific element. Here's a look at the syntax for writing inline style.

<h1 style="color: blue; padding: 20px; text-decoration: underline;">Example Heading</h1>

Example Heading


The above code would just cause the heading to render with blue underlined text and 20 pixels of padding on all sides. There are very few instances where inline styles should be used. In nearly all cases you should avoid using inline css and the styles are added in the style sheet.

Internal Stylesheets:
In the example metioned in the first section we used internal stylesheets. An internal stylesheet is a block of CSS added to an HTML doc. head element. The style element is used between the opening and closing head tags, and all CSS declarations are added between the style tags.

| <head>
| <style>
|   h1{
|     color: blue;
|     padding: 20px;
|     text-decoration: underline;
|   }
| </style>
| </head>
| <body>
| <h1>Example Heading</h1>
| </body>

Example Heading


The above code generates the same results as the inline styles. However, the benefit of using internal stylesheets rather than inline styles is that all h1 elements on the page will be affected by the styles.

External Stylesheets:

External stylesheets are documents containing nothing other than CSS statements. The rules defined in the document are linked to one or more HTML documents by using the link tag within the head element of the HTML document.

To use an external stylesheet. First create the CSS Document.

| /************************
| Save with a name ending in .css such as style.css
| *************************/
|   h1{
|     color: blue;
|     padding: 20px;
|     text-decoration: underline;
|   }

Now that you have an external stylesheet with some styles, you can link it to an HTML document using the link element.

| <head>
| <link rel="stylesheet" type="text/css" href="style.css" >
| </head>
| <body>
| <h1>Example Heading</h1>
| </body>

When this HTML document is loaded the link tag will cause the styles in the file style.css to be loaded into the webpage. As a result, all level 1 heading elements will appear with blue text, underlined, and with 20 pixels of padding applied to every side.
