LINKING WEB PAGES:


LINKING EXTERNAL WEB-PAGES:

URL :

Every website on the Web has a unique address. This address is known as Uniform Resource Locator (URL). Example of URL is as follows:

| Codemugg Homepage URL: "http://codemugg.com/"

HYPERLINK :

The <a> Tag : The anchor element is used to create hyperlinks between a source anchor and a destination anchor. The source is the text, image, or button that limks to another resource and the destination is the resource that the source anchor links to.

ANCHOR ATTRIBUTES :

There are few attributes to create functional hyperlinks. These attributes are href and target. An href can do a lot more than just link to another website.

  - It can be used to link directly to any element on a web page that has been assigned an id.
  - It can be used to link to a resource using a protocol other than http.
  - It can be used to link run a script.

| <a href="http://codemugg.com/"> Codemugg Homepage </a>
|   Codemugg Homepage


Adding a target attribute and _blank attribute to the link will tell the browser to open the link in a new (blank) browser tab or window to render that code in the browser, when a visitor clicks the link which will open in a new tab.

| <p> Welcome to HTML and CSS course by <a href="http://codemugg.com/" target="_blank">CODEMUGG (TARGET)</a>..! <br> Learning has never been this fun..! </p>
| Welcome to HTML and CSS course by CODEMUGG (TARGET)..!
| Learning has never been this fun..!


CSS LINKS :


An HTML document renders the link as following:

  - An unvisited link is underlined and blue.
  - A visited link is underlined and purple.
  - An active link is underlined and red.

Let's create a link and check it's default properties.

|   <a href="#"> LINK </a>
| LINK
Now, Let's try changing the color of link when user have not visited it.

|   a:link {
|     color: c10487;
|   }
| LINK

The following code changes the color of link after the user have visited the link.
|   a:visited {
|     color: #388607; /* Green_Color */
|   }
| EXAMPLE_LINK
You can even change the color of the link when the user hovers their mouse_pointer over the links.

|   a:hover {
|     color: #4076e2; /* Blue_Color */
|   }
| EXAMPLE_LINK
