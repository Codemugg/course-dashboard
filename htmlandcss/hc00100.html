WHAT IS HTML..?


HTML stands for Hypertext Markup Language , is the most commonly used markup language for creating web-pages . Rather, a web page is an HTML document accessible on the World Wide Web.

HTML describes the structure of a Web page. HTML consists of a series of elements, which tells the browser how to display the content.

WHAT IS CSS..?


CSS stands for Cascading Style Sheets and it is the language used to style the visual presentation of web pages.
CSS tells web browsers how to render the different parts of a web page.

There are three ways of linking a CSS file to an Html document.

  • Inline CSS
  • External CSS
  • Internal CSS


The best thing about HTML and CSS is that you don't have to setup any special environment or install any specific ide's. All you need is a text editor and that's all...!
