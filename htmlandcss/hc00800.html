HTML IMAGES:


1. Inserting Images in HTML Document:

In HTML, images are defined with the <img> tag. The syntax of <img> tag can be given with:

| <img src="path_of_image" alt="Some_text" />

src : The 'src' attribute contains a path pointing to the image which is to be inserted into the page.

alt : The 'alt' attribute is a mandatory attribute which specifies an alternate text for an image, if the image cannot be displayed.

| <head>
| </head>
| <body>
| <img src="Nature.png" alt="Nature_Pic" />
| </body>


Now, Let's use some css styling to modify our image:

CSS IMAGE:


RESPONSIVE IMAGE

If you want an image to scale down if it has to, but never scale up to be larger than its original size, add the following:

| img{
|   max-width: 100%;
|   height: auto;
| }

IMAGE BORDER

You can set a border around an image by using border property.

| img{
|   border: solid #000000 5px;
| }


ROUNDED IMAGE

You can use border-radius property to make rounded images.

| img{
|   border-radius: 50px;
| }
THUMBNAIL IMAGE

You can also make a thumbnail image using CSS properties.

| img{
|   border: solid 5px #ff4949;
|   border-radius: 5px;
|   width: 200px;
| }

DISPLAYING VIDEOS ON WEB PAGES:


To display a video on the web page, you can simply use a <video> tag.

|   <video src="path_of_video"></video>
You can use following code to display a video.

|   <video src="city.mov" controls="controls" width="600" height="400 "></video>
