TEXT LEVEL TAGS (Part - I):


The text-level tags are used for formatting text in an html documents.

BOLD :

Text that appears within <b>...</b> element, is displayed in bold.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     <b> Bold Text </b>
|  </body>
| </html>

OUTPUT:

|   Bold Text


ITALIC :

Text that appears within <i>...</i> element, is displayed in italic.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     <i> Italic Text </i>
|  </body>
| </html>

OUTPUT:

|   Italic Text


UNDERLINE :

Text that appears within <u>...</u> element, is displayed with underline.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     <u> Underlined Text </u>
|  </body>
| </html>

OUTPUT:


Underlined Text



SUPERSCRIPT :

The content of a <sup>...</sup> element is written in superscript.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     This is normal text. <sup> This is text written in superscript. </sup>
|  </body>
| </html>

OUTPUT:


  This is normal text. This is text written in superscript.



SUBSCRIPT :

The content of a <sub>...</sub> element is written in subscript.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     This is normal text. <sub> This is text written in subscript. </sub>
|  </body>
| </html>

OUTPUT:


  This is normal text. This is text written in subscript.



DIV TAG :

The <div>...</div> elements allows to group together, several elements to create sections or subsections of pages.

| <!DOCTYPE html>
| <html>
|  <head>
|    <title>First_Page</title>
|  </head>
|  <body>
|     <div> This is the div section. </div>
|  </body>
| </html>

OUTPUT:


This is the div section.



TEXT LEVEL TAGS (Part - II):


HTML entities were described in the previous chapter.

Many mathematical, technical, and currency symbols, are not present on a normal keyboard.

To add such symbols to an HTML page, you can use the entity name or the entity number (a decimal or a hexadecimal reference) for the symbol.
