HTML AND CSS COURSE FOR BEGINNERS..!

HTML TABLES:


In HTML, table is represented with the <table> tag. Each table row is defined with the <tr> tag. A table heading is defined with <th> tag. By default, table headings are bold and centered. A table data/cell is defined with the <td> tag.

<table>...</table> :
Indicates a table. The tags surround the entire table.

<tr>...</tr> :
Indicates a table row. The tags surround the entire row.

<td>...</td> :
Indicates table data or a cell. The tags surround the entire cell.

<th>...</th> :
Indicates table Heading.

Let's create a basic table using above defined tags.

| <head>
| </head>
| <body>
| <table>
|   <tr>
|     <th>Enrollment_No</th>
|     <th>Student_Name</th>
|   </tr>
|   <tr>
|     <td>18456301</td>
|     <td>Tim</td>
|   </tr>
|   <tr>
|     <td>18456302</td>
|     <td>Sarah</td>
|   </tr>
|   <tr>
|     <td>18456303</td>
|     <td>Nick</td>
|   </tr>
| </table>
| </body>
Enrollment_No Student_Name
18456301 Tim
18456302 Sarah
18456303 Nick

TABLE SUBCOMMANDS:


BORDER :

The border attribute draws a border around the cells. Following code shows the example of table borders.

| table {
|   border: solid black 2px;
| }
Enrollment_No Student_Name
18456301 Tim
18456302 Sarah
18456303 Nick

The following code creates a border around each row.

| table {
|   border: solid black 2px;
| }
| table tr{
|   border: solid black 2px;
| }
Enrollment_No Student_Name
18456301 Tim
18456302 Sarah
18456303 Nick
The following code creates a border around each row and column.

| table tr{
|   border: solid black 2px;
| }
| table td, th{
|   border: solid black 2px;
| }
Enrollment_No Student_Name
18456301 Tim
18456302 Sarah
18456303 Nick

Background Table
Now, Let's try changing the background color of table.

| table tr{
|   border: solid black 2px;
|   background-color: rgb(206, 206, 255);
| }
| table td, th{
|   border: solid black 2px;
| }
