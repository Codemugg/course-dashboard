PYTHON COURSE FOR BEGINNERS.

MODULES:


After some time your code starts to get more complex with lots of functions and variables.
To make it easier to organize the code we use Modules.
A well-designed Module also has the advantage of being reusable, so you write code once and reuse it everywhere.
You can write a module with all the mathematical operations and other people can use it.
And, if you need, you can use someone else's modules to simplify your code, speeding up your project.
In other programming languages, these are also referred to as libraries.

USING A MODULE:


To use a module we use the import keyword. As the name implies we have to tell our program what module to import.
After that, we can use any function available in that module. Let's see an example using the math module.
First, let's see how to have access to a constant, Euler's number.

| # Euler's number
| import math
|
| print(math.e)
|
| 2.718281828459045
In this second example, we are going to use a function that calculates the square root of a number.
It is also possible to use the as keyword to create an alias.

| # Euler's number
| import math as m
|
| print(m.e)
|
| 2.718281828459045
Finally, using the from keyword, we can specify exactly what to import instead of the whole module and use the function directly without the module's name.

This example uses the floor() function that returns the largest integer less than or equal to a given number.

| from math import floor
|
| print(floor(9.283))
|

| 9

IMPORTING BUILT-IN MODULES:


Python comes with many built-in modules.
Earlier in the above examples we used the math module, to calculate floor value of a float number and to find the square root of an integer number.
Let's take a look over the basic built-in modules in python which are mostly used by programmers.

STATISTICS
This module is widely used by programmers to perform statistical operations on a give sample data, mostly in the form of lists.
Mean
A mean() function is used to calculate the mean of a given list of numbers.
| import statistics as stats
| List_of_numbers = [10,23,15,28,11,20]
| print(stats.mean(List_of_numbers))

17.833333333333332
Mode
A mode() function returns the element from a list, which has appeared multiple times..
| import statistics as stats
| List_of_numbers = [10,23,15,28,11,28,20]
| print(stats.mode(List_of_numbers))

| 28
Median
The function median() returns the median of a list.
| import statistics as stats
| List_of_numbers = [10,23,15,28,11,20]
| print(stats.median(List_of_numbers))

| 17.5
Stdev
The function calculates the standard deviation on a list of sample numbers.
| import statistics as stats
| List_of_numbers = [10,23,15,28,11,20]
| print(stats.stdev(List_of_numbers))

| 7.521398046336104


RANDOM

At times, when you have to generate a random number in a program, you can use the Random module to do so.
The Random module consists of many built-in functions.
Let's take a look over them.

random()
The random() function generates any random float number between the range 0.0 to 1.0.
| import random
| print(random.random())

| 0.8947586965842
randint()
The randint() function generates a random number, which lies between the specified range.
| import random
| print(random.randint(0, 15))

| 0
choice()
A choice() function returns any random element from a given sequence of strings.
| import random
| Languages = ['C++','Python','Java']
| print(random.choice(Languages))

| Python
