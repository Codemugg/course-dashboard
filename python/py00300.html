PYTHON COURSE FOR BEGINNERS.

PYTHON DATA TYPES AND THEIR METHODS:


You have already learnt about the basics of all data types in python.

Now, You’ll study all of them in detail along with the methods associated with them.

Integer, Float and Complex are the simplest data types, they are not associated with any methods.

So, let's move on to Strings.

STRINGS :


As mentioned before, Strings are collections of different characters.
There are many String manipulation techniques, Let's study those techniques in detail.

String Concatenation :
Concatenation is when you have two or more strings and you want to join them into one. You can use the ‘+’ operator to do so.

| string1 = 'Happy'
| string2 = 'Coding'
| res_string = string1 + string2
| print(res_string)
| Happy Coding
The above code, adds both the string using a space and stroes the newly created string in the final_string, which then gets printed on the console.

The string type has many built-in methods that let us manipulate them.
You will study how some of these methods work.

Len():

The len() function returns the length of a string.
| string = 'This is a string.'
| print(len(string))
| 17
The length of the string “This is a string.” is 17. Hence, the output of code is 17.

replace() :

The replace() method replaces a part of the string with another.
For example, let's replace 'string' for 'sentence'.

| string = 'This is a string.'
| print(string.replace('string', 'sentence'))
| This is a sentence.
After replacing, the output is ‘This is a sentence’.

upper() :

The upper() method converts all the characters of a string to Uppercase.

| string = 'Happy Coding!'
| print(string.upper())
| HAPPY CODING!
The upper() method turns the case of string: ‘Happy Coding!’ to Uppercase: ‘HAPPY CODING!’

lower() :

Unlike the upper() method, the lower() method converts all the characters of a string to Lowercase.

For example,
| string = 'Happy Coding!'
| print(string.lower())
| happy coding!
The lower() method turns the case of string: ‘Happy Coding !’ to Lowercase: ‘happy coding !’

TUPLES:


A tuple is similar to a list: it's ordered, and allows repetition of items.
There is just one difference: a tuple is immutable.
Immutability, if you remember, means you can't change a tuple after its creation.
If you try to add an item or update one, for instance, the Python interpreter will show you an error.

Initializing a tuple:

Following is an example of empty tuple:

| Scores = ()
Following is an example of tuples with initial values:

| Scores = (90.0,93.0,85.33)
| print(Scores)
| (90.0, 93.0, 85.33)
Adding an element in a Tuple:

As tuples are not mutable. You’ll get an error if you try adding a value in a tuple.

| Scores = (90.0,93.0,85.33)
| Scores.add(55.3)
| print(Scores)
| Traceback (most recent call last):
| File "c:\Users\owner\CodeMuggCourse\course-dashboard-main\python\py.py", line 2, in module
| tup.add(45)
| AttributeError: 'tuple' object has no attribute 'add'

Updating an element in a Tuple :

Updating an item will also return an error.
But there is a trick: you can convert into a list, change the item, and then convert it back to a tuple.

| Scores = (90.0,93.0,85.33)
| Marks = list(Scores)
| Marks[1] = 89.00
| Result = tuple(Marks)
| print(Result)
| (90.0, 89.0, 85.33)
Deleting an element from a Tuple :

For the same reason you can't add an item, you also can't delete an item, since they are immutable.

Retrieving from a Tuple :

You can use the index to reference the item.

| Scores = (90.0,93.0,85.33)
| print(Scores)
| print(Scores[2])
| (90.0, 89.0, 85.33)
| 85.33

Let’s implement a program to check whether a particular element is present in a tuple or not:
| Subjects = ('Mathematics','Language','Physics')
| if 'Mathematics' in Subjects:
|   print("Mathematics is present in Subjects.")
| else:
|   print("Mathematics is not present in Subjects.")
| Mathematics is present in Subjects.
As the mathematics is in the tuple, we get the output as ‘Mathematics is present in Subjects.’

Now, let's move on to lists.

LISTS:


A list has its items ordered and you can add the same item as many times as you want. An important detail is that lists are mutable. Mutability means you can change a list after its creation by adding items, removing them, or even just changing their values.

Initializing a List:

Empty List:

| Scores = []
List with initial values:

| Scores = [90.0, 93.0, 85.33]
| print(Scores)
| [90.0, 93.0, 85.33]
Adding an element in a List ::

To add an item at the end of a list, you can use append() method.

| Books = ['Mathematics', 'Language', 'Physics']
| Books.append('Python')
| print(Books)
| ['Mathematics', 'Language', 'Physics', 'Python']


To add an item at a specific position in a list, you can use insert() method.
| Books = ['Mathematics', 'Language', 'Physics']
| Books.insert(1, 'Python')
| print(Books)
| ['Mathematics', 'Python', 'Language', 'Physics']


In the above code, the first argument is an integer which represents the index of the element that is to be added into the list.
Updating in a List :

To update a list you just have to specify the position of the item to update and set the new value

| Books = ['Mathematics', 'Language', 'Physics']
| Books[1]='Python'
| print(Books)
| ['Mathematics', 'Python', 'Physics']


Deleting from a List :

You can use the remove() method to delete the item given as an argument.

| Books = ['Mathematics', 'Language', 'Physics']
| Books.remove('Language')
| print(Books)
| ['Mathematics', 'Physics']

You can use the clear() method to remove all the elements from a list.
| Books = ['Mathematics', 'Language', 'Physics']
| Books.clear()
| print(Books)
| []


Retrieving from a List :

You can use the index to reference the item.
Remember that the index starts at 0.
So to access the second item use the index 1.


| Books = ['Mathematics', 'Language', 'Physics']
| print(Books[2])
| Physics

Let’s implement a program to check whether a particular element is present in a list or not:
| books = ['Mathematics','Language','Physics']
| if 'Mathematics' in books:
|   print("Mathematics is present in Subjects.")
| else:
|   print("Mathematics is not present in Subjects.")
| Mathematics is present in Subjects.
As the mathematics is in the list, we get the output as ‘Mathematics is present in Subjects.’

SETS:


Sets don't guarantee the order of items and are not indexed.
A key point when using sets: they don't allow repetition of an item.

Initializing a SET:

Empty Set:

| Scores = set()
Set with initial values:

| Names = {'Tim','Nick'}
| print(Names)
| {'Tim', 'Nick'}
Adding in a Set :

You can use the add() method to add one item to the set.

| Names = {'Tim','Nick'}
| Names.add('Sarah')
| print(Names)
| {'Tim', 'Nick', 'Sarah'}


You can also use the update() method to add multiple items at once.
| Names = {'Tim','Nick'}
| Names.update(['Sarah', 'John'])
| print(Names)
| {'Tim', 'Nick', 'Sarah', 'John'}


Sets do not allow repetition, so if you try to add 'Tim' again, nothing changes.
| Names = {'Tim','Nick'}
| Names.add('Tim')
| print(Names)
| {'Tim', 'Nick'}


Updating in a Set :

Items in a set are not mutable. You have to either add or delete an item.

Deleting from a Set :

Similar to lists, You can delete an element in a set using remove() method.

| Books = {'Mathematics', 'Language', 'Physics'}
| Books.remove('Language')
| print(Books)
| {'Mathematics', 'Physics'}

And you can also use the clear() method to remove all the elements from a set.

Let’s implement a program to check whether a particular element is present in a list or not:
| books = {'Mathematics','Language','Physics'}
| if 'Mathematics' in books:
|   print("Mathematics is present in Subjects.")
| else:
|   print("Mathematics is not present in Subjects.")
| Mathematics is present in Subjects.
As the mathematics is included in the set, we get the output as ‘Mathematics is present in Subjects.’

DICTIONARIES:


The dictionary doesn't guarantee the order of the elements and it is mutable.
One important characteristic of dictionaries is that you can set your customized access keys for each element.

Initializing a Dictionary:

Empty Dictionary:

| Phone_bk = {}
Dictionary with initial values:

| Phone_bk = {'Sam':'98653*****','Nick':'98**56**74'}
| print(Phone_bk)
| {'Sam':'98653*****', 'Nick':'98**56**74'}
Adding in a Dictionary :

To add a value to a dictionary, you can use the following syntax:

| Phone_bk = {'Sam':'98653*****','Nick':'98**56**74'}
| Phone_bk['Sarah'] = '**78**85**' | print(Names)
| {'Sam':'98653*****','Nick':'98**56**74', 'Sarah': '**78**85**'}


Updating a Dictionary :

To update the value of key, you just have to use the above syntax which we used to add a value.
The only difference is, the key_value, should already be present in the dictionary.
| Phone_bk = {'Sam':'98653*****','Nick':'98**56**74', 'Sarah': '**78**85**'}
| Phone_bk['Sarah'] = '9178**85**'
| print(Names)
| {'Sam':'98653*****','Nick':'98**56**74', 'Sarah': '9178**85**'}


Deleting a record from a dictionary :

Lets remove ‘Sam’ from the Phone_bk.

| Phone_bk = {'Sam':'98653*****','Nick':'98**56**74'}
| Phone_bk.pop('Sam')
| print(Names)
| {'Nick':'98**56**74', 'Sarah': '9178**85**'}

The above code removes Sam’s contact information from the phone_bk.
And similar to lists and sets, we use the clear() method to remove all the records from a dictionary.

Accessing an element in a dictionary :

A value in a dictionary can be accessed using its key name.

| Phone_bk = {'Sam':'98653*****','Nick':'98**56**74'}
| Record = Phone_bk['Sam']
| print(Record)
| '98653*****'

A program to check whether a key is present in a record or not.

| Phone_bk = {'Sam':'98653*****','Nick':'98**56**74'}
| if 'Nick' in Phone_bk:
|   print("Nick's contact is available.")
| else:
|   print("Nick's contact is not available.")
| Nick's contact is available.
