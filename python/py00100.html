WHAT IS PYTHON..?


Python is one of the most popular and an open-source programming language created by Dutch programmer Guido van Rossum, in the late 1980’s. It is named after the British sketch comedy group Monty Python. It is the easiest language to get started with if you are new to programming.

WHERE IS IT USED..?


System scripting: It's a great tool to automate everyday repetitive tasks.
Data Analysis: It is a great language to experiment with and has tons of libraries and tools to handle data, create models, visualize results and even deploy solutions. This is used in areas like Finance, E-commerce, and Research.
Web Development: Frameworks like Django and Flask allow the development of web applications, API's, and websites.
Machine Learning: Tensorflow and Pytorch are some of the libraries that allow scientists and the industry to develop and deploy Artificial Intelligence solutions in Image Recognition, Health, Self-driving cars, and many other fields.

Let’s get started by installing Python3 on your current machines.

INSTALLING PYTHON:


Installing python is the first step towards becoming a python programmer. There are various ways of downloading and installing python on your systems. You can download the official distribution of Python from Python.org.
Note: If you are using Mac OS or Linux OS you already have python installed on your systems.
We’re going to use the Python 3.9.0 version throughout this course.
To check the current version of python on your system, enter the following command in your terminal:

| C:\Users\owner>python -V
Which gives the following output:

| python 3.9.0
If your result is something similar to 'Python 3.x.y', for instance, Python 3.9.0 or 3.9.1, then you are set to go.

If not, follow the following instructions according to your Operating System.

Installing Python 3 on Windows:
1. Go to Python.org.
2. Download the latest version.
3. After the download, double-click the installer.
4. On the first screen, check the box indicating to "Add Python 3.x to PATH" and then click on "Install Now".
5. Wait for the installation process to finish until the next screen with the message "Setup was successful".
6. Click on "Close".

Installing Python 3 on Mac:
1. Install XCode from the App Store.
2. Install the command line tools by running the following in your terminal.

| xcode-select --install
I recommend using Homebrew.

Go to Homebrew and follow the instructions on the first page to install it.
After installing Homebrew, run the following brew commands to install Python 3.

| brew update brew install python3
Homebrew already adds Python 3 to the PATH, so you don't have to do anything else.

Installing Python 3 on Linux:

To install using apt, available in Ubuntu and Debian, enter the following:

| sudo apt install python3
To install using yum, available in RedHat and CentOS, enter the following:

| sudo yum install python3
As you have installed python on your system, lets learn about python development environment.

THE INTERACTIVE SHELL.


Python comes with an In-built IDLE, which stands for Integrated Development Environment. IDLE is the place where you will type your python code.
Once you are done with the installation of Python on your system. Search for IDLE in your Explorer (Windows), Finder (Mac), or Nautilus (Linux).
Click on the IDLE icon, this will open up the python interactive shell:

Img 1.1: Interactive Shell

This program is called the interactive shell. You can type Python code directly into the interactive shell, and it will print the results. At the prompt >>> type:

| >>> print(“Hello Codemugg.”)
And press enter.
This will give you following output:

| Hello Codemugg
By the way, we use the print() function to display the message written within quotes, on the screen.

In the programming world, It’s tradition to start your programming journey by running the print Hello program.

So kudos to you..! You just wrote your first python program.

Now, let’s move on to our next topic i.e. Running and Saving python programs.

RUNNING & SAVING PYTHON PROGRAMS.


You can use Interactive shell for quick computations, and writing short programs. But as you move ahead, you are going to write more complex programs.

Using IDLE to run such complex programs isn’t a good choice.

The alternative is simply to use a text editor, type the code, and save the file with a .py extension.
