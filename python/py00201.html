PYTHON COURSE FOR BEGINNERS.

INTRODUCTION TO PYTHON PROGRAMMING:


CONDITIONAL STATEMENTS:


Conditionals are one of the cornerstones of any programming language. They allow you to control the program flow according to specific conditions. A conditional Statement is like a control structure: a block of code which makes decisions analyzing the values of variables.
The keywords ‘if’, ‘ else ’ and ‘ elif ’ are used to implement conditional statements in python. It allows the execution of a particular block of code conditionally.
Following pseudocode displays the syntax of conditional statements:

1. | if ( CONDITION_IS_TRUE ):
2. |
3. |   Execute Following block of code
4. |   BLOCK 1
5. |
6. | else:
7. |
8. |   Execute Following block of code
9. |   BLOCK 2
The above pseudocode explains that you can define two conditional statements simultaneously. If the condition on line 2 turns out to be true, then the block of code written in BLOCK 1 is executed, otherwise the control is passed on to the BLOCK 2.

if statement:

The block of code written inside the if statement is only executed, if the condition at the start is true.
The general form of a if statement is:

1. | if ( CONDITION_IS_TRUE ):
2. |
3. |   Execute Following block of code

The condition contains some logic that returns a boolean, and the statement is executed only if the return is True.

You’ll understand the use of if statement using following example:

| Cost_apples = 50.00
| Cost_mangoes = 45.00
| if (Cost_apples > Cost_mangoes):
|   print('Apples are costly.')
| Apples are costly.
As the condition after the if statement is evaluated to true. The program executes the statement after the ‘ if ’ condition and prints ‘ Apples are costly. ’

if, else Statement:

In our last example, the program only does something if the condition returns True.

But we also want it to do something if it returns False or even check a second or third condition if the first one went wrong.

In the following code we’ll change the condition to Cost_apples < Cost_mangoes. Hence, the first expression will return False since Apples are costlier than mangoes, and the program will execute the code written in else block.

| Cost_apples = 50.00
| Cost_mangoes = 45.00
| if (Cost_apples < Cost_mangoes):
|   print('Mangoes are costly.')
| else:
|   print('Apples are costly.')
| Apples are costly.
As the program executes the block of code written after ‘ else ’ is displayed on the console.

elif Statement:

Let's change the cost of mangoes to 50.00.

Now, to check whether the price of apples and mangoes is equal, you can include an ‘elif ’ condition in your program

In the following code we’ll change the condition to Cost_apples < Cost_mangoes. Hence, the first expression will return False since Apples are costlier than mangoes, and the program will execute the code written in else block.

| Cost_apples = 50.00
| Cost_mangoes = 45.00
| if (Cost_apples < Cost_mangoes):
|   print('Mangoes are costly.')
| elif (Cost_apples == Cost_mangoes):
|   print('Apples and Mangoes cost the same.')
| else:
|   print('Apples are costly.')
| Apples and Mangoes cost the same.
The condition in the ‘ elif ’ is met. Since, the cost of both the fruits are equal the program will print ‘Apples and Mangoes cost the same. ’ on the console.

Note: You can have as many elifs as you want, just put them in sequence.

Nested conditions:

There are times when you have to consider more than a single condition to perform a particular task. You can use Nested if statements or Nested if else statements.

For example,
Lets develop a program which checks whether an year is a leap year or not.

| # You can try executing the program for different values of year.
| year = 2016
| if (year%4==0):
|   if (year%100==0):
|     if (year%400==0):
|       print('This is a leap year.')
|     else:
|       print("It's not a leap year.")
|   else:
|     print("This is a leap year.")
| else:
|   print("It's not a leap year.")
| This is a leap year.
The above code checks whether a year is a leap year or not. We have used three if statements which are all associated with an else statement.

Ternary Operator:
The ternary operator is a one-line if statement.
It's very handy for simple conditions.

This is how it looks:

| ( EXPRESSION ) if ( CONDITION ) else ( EXPRESSION )
Lets execute a python program to evaluate the cost of fruits, just like we did earlier, using ternary operators.

| Cost_apples = 50.00
| Cost_mangoes = 45.00
| print('Apples are not costly.') if Cost_apples < Cost_mangoes else print('Apples are costly.')
| Apples are costly.
In the above code, the condition ( Cost_apples < Cost_mangoes ) is evaluated to False. Hence, the statement mentioned prior to if condition are not executed.

LOOPS:


A loop continuously executes a specific block of code until a particular condition is satisfied.
In this section, you’ll learn about various kinds of loops and their implementation in the python programming.
There are two types of loops: for and while. You will learn about while loops in the next section.

for Loops:

A for loop is used to iterate through an iterable. This process is called iterating. You can use a for-loop to define instructions that execute once for every item in an iterable, and you can access and manipulate each item in the iterable from within the instructions you defined.

| for ELEMENT in COLLECTION:
|   ( STATEMENTS )
For example,
You can use a for loop to access all the characters in a string.

| string = ('CODEMUGG')
| for char in string:
|   print(char)
| C
| O
| D
| E
| M
| U
| G
| G
In the above code, using a for loop we traverse through each character in a string and print them one at a time.

You can use for-loops to change the items in a mutable iterable, like a list:

| my_list = ['Welcome','to','Codemugg']
| for words in my_list:
|   print(words.upper())
| WELCOME
| TO
| CODEMUGG
In the above program, we used a string method i.e. upper() which converts all the characters of a string to uppercase.

What if you want to iterate through a range of numbers.. ?



For that purpose we have an inbuilt range function called as range() .

Range() :
The range function is widely used in for loops because it gives you a simple way to list numbers.
The following code will loop through the numbers 0 to 5 and print each of them.

| for num in range(5):
|   print(num)
| 0
| 1
| 2
| 3
| 4
The range() function takes a lower bound and an upper bound as its arguments. But if you just pass a single number as an argument like in the above code, it is considered as upper limit. And the lower limit is set to 0 by default, that’s why the numbers that range between 0 to 5 are displayed in the output.

Finally, it is also possible to set a step.
Here we are starting at 10 and incrementing by 2 until 20, since 20 is the upper limit, it is not included.

| for num in range(10,20,2):
|   print(num)
| 10
| 12
| 14
| 16
| 18
If you want to terminate a loop in between its execution, you can use a break statement for it.

Break :
A break keyword is used to stop the execution of a loop.

| my_country = ['India','Argentina','Canada','Brazil']
| for country in my_country:
|   print(country)
|   if country == 'Canada':
|     break
| India
| Argentina
| Canada
The above code illustrates the use of break statement. The countries in the loop are printed until the if condition is evaluated to true. When the if condition is statisfied the loop is terminated.

If you just want to skip a particular element from a list and continue the execution of the next statement, you can use a continue keyword to skip the element.

Continue :

A continue keyword is used to skip a specific element from a list.
Let’s try using a continue keyword in the above example:

| my_country = ['India','Argentina','Canada','Brazil']
| for country in my_country:
|   if country == 'Canada':
|     continue
|   print(country)
| India
| Argentina
| Brazil
The above program prints all the countries in list which doesn’t statisfies the if condition.
When the iterator reaches to ‘ Canada ’ the block of code after the condition is eliminated and the execution is continued from next country i.e. ‘ Brazil ’.

while loop :

The block of code written inside a while loop is only executed if the condition at the start of the loop is satisfied.
The syntax of a while loop is shown below:

| while ( CONDITION ):
|   ( STATEMENTS )
The loop will continue executing until the condition is True.
Following code is an example of implementing a while loop:

| # Printing square of numbers
| number = 1
| while ( number <= 5 ):
|   print(number, ' = ', number**2)
|   number += 1
| 1 = 1
| 2 = 4
| 3 = 9
| 4 = 16
| 5 = 25
The code executes till the condition number <= 5 is met. Once, the value of number becomes 6, the loop is terminated.

else block :
When the condition at the start of the while loop returns False, the else block will be executed.
| # Printing square of numbers
| number = 1
| while ( number <= 5 ):
|   print(number, ' = ', number**2)
|   number += 1
| else:
|   print("Program executed..!")
| 1 = 1
| 2 = 4
| 3 = 9
| 4 = 16
| 5 = 25
| Program executed..!
To break out of a while loop in Python:
Similar to the for loop, you can simply use the break keyword, and the loop will stop its execution.

| # Printing square of numbers
| number = 1
| while ( number <= 5 ):
|   if number == 4:
|     break
|   print(number, ' = ', number**2)
|   number += 1
| else:
|   print("Program executed..!")
| 1 = 1
| 2 = 4
| 3 = 9
The loop runs normally, and when number reaches 4 the if statement evaluates to True and the break command is called. This finishes the loop before the squared value of the numbers 4 and 5 are calculated and else the number doesn’t get incremented further, the while condition is always evaluated to true and the else block is never executed.

To skip the execution of a particular statement of a while loop in Python: Similar to the for loop, you can simply use the continue keyword, and the loop will skip the execution and continue from the next condition.

| # Printing square of numbers
| number = 1
| while ( number <= 5 ):
|   number += 1
|   if number == 4:
|     continue
|   print(number, ' = ', number**2)
| else:
|   print("Program executed..!")
| 1 = 1
| 2 = 4
| 3 = 9
| 5 = 25
| Program executed..!
The program always checks if 4 is the current value of the number. If it is, the square of 4 won't be calculated and the continue will skip to the next iteration when the value of number is 5.
